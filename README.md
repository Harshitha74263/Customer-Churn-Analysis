# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn patterns and business insights using Power BI.

## Objectives
- Identify factors causing customer churn
- Analyze customer demographics
- Track churn rate and retention trends
- Improve customer retention strategies

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Dashboard Features
- KPI Cards
- Interactive Slicers
- Churn Trend Analysis
- Customer Segmentation
- Revenue Analysis

## Key KPIs
- Total Customers
- Churn Rate
- Active Customers
- Monthly Revenue
- Retention Rate

## Business Insights
- Customers with higher monthly charges churn more
- Month-to-month contracts have highest churn
- Fiber optic users show higher churn rate
